# 红书工厂

- English slug: `xiaohongshu-content-engine`
- Status: `proposed`
- Type: content distribution experiment candidate
- Current phase: project definition only

## Positioning

红书工厂围绕 Codex、AI 自动化、AI 自养实验，生产小红书内容草稿，用于验证市场兴趣、获取流量，并最终引流到图书馆模板包或课件。

This is not an approved build project. Codex may define the project and future validation needs, but may not generate the first 20 posts or automate publishing until separately approved.

## Core hypothesis

小红书比独立网站更容易打开 0 到 1，因为内容可以直接获得推荐、搜索、收藏、评论和私信反馈。

## Target users

- 想学习 AI 但不知道从哪里开始的人
- 想用 Codex 提高工作效率的人
- 想用 AI 做副业的人
- 自由职业者
- 独立开发者
- 被“AI 会不会赚钱”困扰的人

## Content directions

1. Codex 是什么
2. 如何配置 Codex
3. Codex 可以做什么
4. 不会用 Codex 会落后吗
5. 我让 AI 尝试自己赚钱
6. AI 自养实验日志
7. 普通人如何用 AI 做项目
8. 用 AI 搭一个自动赚钱系统的全过程

## Phase 1 goal

Within 7 days after approval:

- Generate 20 Xiaohongshu content drafts
- Chairman manually selects and publishes 7 posts
- Record performance data for each published post

Required data fields per post:

- Published at
- Title
- Topic
- Views
- Likes
- Saves
- Comments
- Direct messages
- Whether anyone requested materials
- Whether anyone asked about price

## Monetization path

### Early stage

- Free resource pack lead-in
- Low-price template pack
- Writing fees or collaboration opportunities

### Middle stage

- Codex setup course materials
- AI automation project course materials
- 图书馆 template pack

### Later stage

- Cohort program
- Consulting service
- Enterprise automation package

## Constraints

- Codex must not automatically publish to Xiaohongshu
- Codex only generates drafts and topic ideas after approval
- Every post must be manually confirmed by Chairman before publishing
- Do not use fake cases
- Do not promise guaranteed income
- Do not create exaggerated fear-based marketing
- Do not impersonate official accounts or platforms
- Do not automatically message users
- Do not collect user private information
- Do not bypass platform rules

## Stop criteria

Pause the project and review titles, topics, and target users if 7 posts are published within 14 days and all of the following are true:

- Total views are below 500
- Total saves are below 10
- Comments or direct messages equal 0
- Nobody requests materials

## Current decision

This project is recorded as `proposed`. It is a second-project candidate only. No code, posting automation, account operation, content batch generation, or publishing workflow is approved in this step.
