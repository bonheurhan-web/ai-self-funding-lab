# Metrics

Raw snapshots and source exports supporting dashboard values belong here.

Use:

```text
metrics/
  YYYY-MM/
    <project-name>/
      traffic.*
      indexing.*
      revenue.*
      cost.*
```

Do not commit credentials, personal data, or exports containing unnecessary user-level identifiers. Canonical definitions are in [`docs/METRICS.md`](../docs/METRICS.md).
