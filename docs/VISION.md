# Vision

AI Self-Funding Lab becomes a portfolio company capable of launching, operating, measuring, and retiring up to 100 small internet businesses with minimal human execution.

## Desired state

- New projects start from one governed template.
- Every deployment, cost, traffic source, and revenue event is attributable to a project.
- Routine testing, deployment, monitoring, reporting, and indexing checks are automated.
- Shared capabilities are reused through documented contracts, not copied ad hoc.
- Projects compete for time and budget using comparable portfolio metrics.
- Failed experiments create reusable evidence instead of hidden sunk cost.

## Non-goals

- Building a single large startup before demand is proven.
- Maximizing code volume, technical novelty, or project count.
- Sharing code merely because two projects currently look similar.
- Keeping projects alive without a measurable path to revenue.
